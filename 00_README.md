# DS / ML / AI Playbook

## Overview

This repository is a **living playbook** covering core foundations and advanced topics across:

- Data Science
- Machine Learning Engineering
- AI Engineering

It is designed to serve multiple purposes:
- On-the-job reference material
- Personal learning and system design reinforcement
- A structured knowledge base suitable for Retrieval-Augmented Generation (RAG)

---

## How This Playbook Was Created

The **structure and organization** of this playbook were **AI-assisted**.

Specifically:
- Folder layout
- Topic taxonomy
- Ordering and progression of concepts
- Separation between foundations, systems, and applied pipelines

These elements were generated collaboratively with AI to ensure:
- Coverage completeness
- Logical progression
- Industry-aligned mental models

---

## Content Authorship & Responsibility

All **technical content**, including:
- Explanations
- Code examples
- Design tradeoffs
- Failure modes
- Heuristics
- Interview framing
- Commentary and conclusions

has been **human-generated, curated, edited, and validated**.

AI was used as an accelerator for:
- Drafting
- Refactoring
- Consistency checks
- Expanding examples

However, **final judgment, correctness, and intent are entirely human-owned**.
